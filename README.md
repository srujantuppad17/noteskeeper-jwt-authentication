# 📝 NotesKeeper - JWT Authentication

A secure full-stack NotesKeeper application built using the MERN Stack. This project implements user authentication using JSON Web Tokens (JWT), protected routes, and complete CRUD operations for notes.

---

## 🚀 Features

- 🔐 User Registration
- 🔑 User Login with JWT Authentication
- 🛡️ Protected Routes
- ➕ Create Notes
- 📖 View Notes
- ✏️ Edit Notes
- ❌ Delete Notes
- 💾 MongoDB Database Integration
- 🎨 Responsive React User Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/srujantuppad17/noteskeeper-jwt-authentication.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=4005
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
